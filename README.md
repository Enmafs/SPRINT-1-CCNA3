# SPRINT-1-CCNA3
La implementación de este laboratorio demuestra la capacidad de implementación de los siguientes criterios:                                                                                                                                                                                                                                                                                                                                 
• Conectividad entre VLANs según política definida.                                                                                                                                                                   
• Redundancia de gateway funcional (HSRP) con failover probado.                                                                                                                                                       
• Convergencia de OSPF y propagación de rutas estáticas predeterminada).                                                                                                                                              
• Accesos restringidos por ACLs según políticas de seguridad diseñada por el equio.                                                                                                                                   
• Implementación en PNETLab.                                                                                                                                                                                          
                                                                                                                                                                                                                      
Más detalles de lo que corre detrás de la red:                                                                                                                                                                        
• OSPF multi área                                                                                                                                                                                                     
• VTP                                                                                                                                                                                                                 
• HSRP                                                                                                                                                                                                                
• Spanning Tree Protocol                                                                                                                                                                                              
• Implementación de ACL                                                                                                                                                                                               
• VLAN                                                                                                                                                                                                                
• VTP                                                                                                                                                                                                                 
• VTP domains
